# AgentHomeAssistant

基于 Apple 生态 + 小米智能家居的个人家庭助理自动化方案

## 现有设备清单

- 米家激光投影仪
- Apple TV 4K
- 小米踢脚线暖风机
- 小米空气净化器
- iPhone (苹果手机)

## 自动化方案 brainstorm

### 1. 观影场景自动化

#### 场景触发条件
- Apple TV 开始播放视频 (通过快捷指令自动化或 HDMI-CEC 检测)
- 手动触发"观影模式"

#### 自动化动作
1. **灯光控制**：关闭主灯，保留氛围灯/小夜灯
2. **投影仪自动开启**：米家投影仪通过蓝牙或米家 API 自动开机，自动切换到 HDMI 输入 (Apple TV)
3. **窗帘关闭**：如果有智能窗帘，自动关闭窗帘
4. **暖风机调节**：根据当前温度自动调整功率，观影过程中保持舒适温度，避免过干过热
5. **空气净化器**：自动调至低速/静音模式，避免噪音影响观影

### 2. 暖风机智能控制

#### 控制逻辑
- 根据室内温度自动启停/调节功率
- 结合天气预报和室温变化，提前预热
- 人体存在感应 (如果添加传感器)：有人在房间才开启，无人自动关闭节能
- 夜间模式：自动降低功率和温度，避免过热影响睡眠

#### 与 Apple 生态集成
- 通过家庭 App (HomeKit) 控制，需要添加网关 (如 Aqara 网关) 将小米设备接入 HomeKit
- 快捷指令：Siri 语音控制 "打开客厅暖风机" "调高温度"

### 3. 空气净化器自动化

#### 自动化逻辑
- 根据空气质量 (PM2.5 传感器数据) 自动调节档位
- 回家前自动开启净化
- 夜间自动切换静音模式
- 滤网寿命提醒：当寿命低于 X% 时发送通知提醒更换

### 4. 投影仪自动化

#### 使用场景自动化
- 观影模式：自动开机 + 切换输入源
- 结束观影：自动关机，恢复灯光和窗帘
- 节能：投影无信号 10 分钟后自动关机

### 5. 集成方案选型

**方案 A: HomeBridge 接入**
- 优点：开源免费，社区支持完善，小米设备插件丰富
- 缺点：需要一台始终运行的树莓派或主机跑 HomeBridge
- 适合：技术玩家，成本低

**方案 B: Aqara 网关 + HomeKit 直连**
- 优点：稳定，官方支持，原生家庭 App 体验
- 缺点：需要额外购买 Aqara 网关，部分小米设备兼容性问题
- 适合：追求稳定，不想折腾的用户

**方案 C: 基于 Home Assistant 的集成**
- 优点：高度自定义，支持几乎所有品牌设备，可以做复杂自动化
- 缺点：配置复杂，需要硬件运行，资源占用较高
- 适合：需要复杂自动化场景，技术能力强

**方案 D: 使用米家 App + 快捷指令对接**
- 优点：无需额外硬件，通过米家 App 已经有的自动化，再用快捷指令调用
- 缺点：自动化逻辑相对简单，跨设备场景联动不够灵活
- 适合：简单场景，不想折腾额外服务

### 6. 推荐架构选型 (初步推荐)

选择 **Home Assistant + HomeBridge** 方案：
1. Home Assistant 作为核心自动化引擎，处理复杂的跨设备场景联动
2. 通过 HomeBridge 小米插件将所有小米设备接入 HomeKit
3. 这样 iPhone + Apple TV 可以原生通过家庭 App 控制，支持 Siri 语音
4. 可以实现非常复杂的场景自动化，比如"观影模式"一键触发所有设备联动

### 7. 后续需要补充的设备 (建议)

- **Aqara 人体存在传感器**：实现人来自动开设备，人走自动关闭节能
- **温湿度传感器**：给暖风机提供更准确的温度数据，自动调节
- **PM2.5 传感器**：如果空气净化器不带，可以额外添加给自动化决策
- **智能开关面板**：实现物理按键 + 自动化双重控制灯光
