# 弱电布线需求文档

> **用途**：供与装修公司沟通使用，确保弱电布线满足智能家居需求
> **更新时间**：2026-06
> **关键节点**：水电进场前必须确认，水电施工中验收

---

## 一、总体要求

### 1.1 布线原则

| 原则 | 说明 |
|------|------|
| 网线规格 | 全屋统一使用 **Cat6 类网线**（超六类 Cat6a 更佳），支持千兆/万兆 |
| 网口数量 | 每个房间至少 **2 个网口**（主口 + 备用） |
| 弱电箱 | 尺寸至少 **400×300×100mm**，预留散热空间和扩展位 |
| 零线预留 | **所有开关底盒必须预留零线（N线）**，这是智能家居的硬性要求 |
| 线管 | 弱电与强电分管走线，间距 ≥ 30cm，避免干扰 |

### 1.2 验收标准

- [ ] 所有网口网络连通性测试通过（用测线仪逐个测试）
- [ ] 所有开关底盒零线通断测试通过
- [ ] 弱电箱尺寸满足设备放置需求
- [ ] 线管无破损，穿线顺畅，预留维修余量

## 三、开关底盒要求（关键）

### 3.1 零线预留（P0 必须项）

**所有开关底盒必须预留零线（N线）！**

| 要求 | 说明 |
|------|------|
| 零线 | 每个开关底盒必须有火线(L)、零线(N)、控制线(回火线) |
| 底盒深度 | 标准 86 盒，深度 ≥ 50mm（推荐 60mm 深底盒） |
| 底盒材质 | 阻燃材质，金属底盒更佳（散热好） |

> **风险提示**：若无零线，只能使用单火版智能开关，存在以下问题：
> - 闪烁/不稳定
> - 小功率灯具无法正常工作
> - 价格更高、型号选择少

## 六、验收检查清单

### 水电施工中验收

- [ ] 所有网口已穿 Cat6 网线，测线仪测试 8 芯全通
- [ ] 所有开关底盒有零线（N线），万用表测试通断
- [ ] 弱电箱尺寸 ≥ 400×300×100mm，内部有 ≥ 4 个电源插座
- [ ] 窗帘盒位置已预留电源插座
- [ ] 天花板投影仪位置已预留电源 + HDMI 线管
- [ ] 弱电与强电线管间距 ≥ 30cm
- [ ] 所有线管无破损，穿线顺畅

### 硬装完成后验收

- [ ] 所有网口面板安装完成，网络连通
- [ ] 所有开关底盒安装完成，零线到位
- [ ] 弱电箱内设备可正常放置
- [ ] 窗帘电机电源可用
- [ ] 投影仪吊装位电源和 HDMI 线管可用
