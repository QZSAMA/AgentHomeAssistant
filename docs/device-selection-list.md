# 智能家居设备选型清单

> **用途**：设备采购前的选型确认，含与绿米/欧瑞博的沟通要点
> **更新时间**：2026-06
> **采购时间**：2026.12 硬装完成后

---

## 一、选型原则

| 原则 | 说明 |
|------|------|
| HomeKit 兼容 | 优先选择原生支持 Apple HomeKit 的设备 |
| HA 集成 | 确认 Home Assistant 有官方或成熟社区集成 |
| 零线版本 | 所有智能开关选择**零火版**，不选单火版 |
| 协议统一 | 优先 Zigbee 3.0 组网，统一网关管理 |
| 固件可升级 | 确认设备支持 OTA 固件升级 |
